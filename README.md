# 🎮 Stone Paper Scissor Game (Python Tkinter)

A simple **Stone-Paper-Scissor game** built using **Python and Tkinter GUI library**.
In this game, the user selects **Stone, Paper, or Scissor**, and the computer randomly chooses one option. The program then determines the winner and displays the result in the GUI.

---

## 📌 Features

* Simple **Tkinter GUI interface**
* **Random computer opponent** using Python's `random` module
* Displays:

  * Player choice
  * Computer choice
  * Game result (Win / Lose / Tie)
* Beginner-friendly Python project

---

## 🛠 Technologies Used

* **Python**
* **Tkinter** (GUI library)
* **Random module**

---

## 🎮 How the Game Works

1. The player clicks one of the buttons:

   * Stone
   * Paper
   * Scissor

2. The computer randomly selects one option.

3. The program compares both choices and displays:

   * Player choice
   * Computer choice
   * Winner of the round

## 📸 Example Output

The GUI window shows:
